《行者上任特雷森，大圣再当弼马温》的话需要额外清理签名栏：

BBCode转HTML：

import bbcode

parser = bbcode.Parser()

parser.add_simple_formatter('h1', '<h1>%(value)s</h1>')
parser.add_simple_formatter('h2', '<h2>%(value)s</h2>')
parser.add_simple_formatter('del', '<span style="text-decoration:line-through;color:#555">%(value)s</span>')
parser.add_simple_formatter('chartradar', '<div>%(value)s</div>') # 虽然没有chartradar，但还是象征性地搞一下。
parser.add_simple_formatter('collapse', '<details><summary>%(collapse)s</summary>%(value)s</details>')

html = parser.format(text)
