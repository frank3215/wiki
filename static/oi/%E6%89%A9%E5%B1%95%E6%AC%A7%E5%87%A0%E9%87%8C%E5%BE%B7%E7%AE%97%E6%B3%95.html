例题：NOI2002 - 荒岛野人。

令b′=a−b⌊ab⌋,a′=bb'=a-b\left\lfloor\frac{a}{b}\right\rfloor, a'=bb′=a−b⌊ba​⌋,a′=b，若

a′x′+b′y′=d a'x' + b'y' = d a′x′+b′y′=d

则

ay′+(x′−⌊ab⌋y′)b=d ay' + \left(x'-\left\lfloor\frac{a}{b}\right\rfloor y'\right)b=d ay′+(x′−⌊ba​⌋y′)b=d

void exgcd(int a, int b, int &d, int &x, int &y) {
    if (b == 0) d = a, x = 1, y = 0;
    else exgcd(b, b%a, d, y, x), y -= (a/b)*x;
}
