用TiddlyWiki进行协作

2021年06月28日 09:53
public TiddlyWiki

热烈欢迎！！

参考Contributing and collaborating using GitHub Desktop。

改进本文内容可以参考如何参与 - OI Wiki。

入门

阅读Github官方入门指南，确保你已经注册了一个Github帐号。

下载Github Desktop并登录。

修改

参考Fork a repo，Cloning a repository to GitHub Desktop。

fork当前已有的repo。

在Github Desktop里clone已经fork好的repo，选择To contribute to the parent project后点击Continue。

到该文件夹的上一层的母文件夹，运行TiddlyWiki服务器端（tiddlywiki wiki --listen），进行你想要的更改，保存并终止（ctrl-C）服务器端。

打开Github Desktop，进行commit操作。

注：如果你的改动不只由你一个人完成，可以增加合著者，见Creating co-authored commits using GitHub Desktop。

合并

参考Creating a pull request from a fork。

在github.com上你自己fork好的的repo中点击Contribute，选择Create pull Request（要点两次）。

等待审核即可！

如果您有直接修改的权限……

参考Managing branches中的Creating a branch。

可以不用fork，clone下来创建一个新的branch，在这个branch上进行您的修改后再提出Pull Request来merge即可。
