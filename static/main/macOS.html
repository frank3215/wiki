Tips：

  • 压缩（.zip）时会创建__MACOSX/文件夹和.DS_Store文件。你可以用Keka来规避这一点。
    • macos - What is __MACOSX folder? - Super User
  • iBooks只支持比较少的格式（.pdf、.epub），你可以用Calibre来查看其他格式的文件，与进行不同格式之间的转换。
    • 它会自动把.epub解压。恢复回普通EPUB的方法可以用这里的ePub Zip-Unzip 64 3.0.zip脚本。来自这个Reddit帖子。
    • 书的位置可能在~/Library/Mobile Documents/iCloud~com~apple~iBooks/Documents。
  • 无法打开.app时，可以试试用find <.app> -type f -exec chmod +x {} \;来加权限。
  • 和Android手机互传文件请用AndroidFileTransfer。
  • 提示“已损坏”时可以使用sudo xattr -r -d com.apple.quarantine /Applications/<应用>.app来绕过
  • 提示“您没有权限打开应用程序”时可以尝试这个。
  • 在 Mac 上使用桌面叠放
  • Mac keyboard shortcuts
  • Mac 基础教程：如何更改你的 Mac 设备名称 - 少数派
